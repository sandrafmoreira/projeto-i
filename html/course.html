Descobre como esta licenciatura pode ser uma mais valia para o teu futuro!
A licenciatura foca-se no design e desenvolvimento de produtos e software. Destaca-se pelo desenvolvimento de competências fundamentais como design de interação, usabilidade e experiência de utilizador, além da prototipagem de plataformas.
A combinação de todas as aptidões é essencial para o mundo de trabalho e nesta área em específico, e é uma característica marcante da licenciatura que contribui para o seu sucesso.

Saídas Profissionais

Desenvolvimento de Produtos Multiplataforma

Especialização na criação de aplicações e softwares que podem ser funcionais em diferentes sistemas operativos e dispositivos.

UI/UX Developer

Desenvolvimento de interfaces de utilizador eficazes e agradáveis, com layouts atraentes e funcionais, com foco na usabilidade e acessibilidade.

Desenvolvimento de plataformas de comércio e negócio eletrónico

Desenvolvimento de interfaces de utilizador amigáveis e intuitivas, bem como uma boa optimização do desempenho de uma plataforma para garantir uma boa experiência ao utilizador.

Conceção e Desenvolvimento de Software e Produtos Web

Responsável por criar aplicações e programas que funcionem em ambiente web, tendo como responsabilidade a análise de requisitos de um cliente, bem como o design do software.

O que é UI?

User Interface (UI) é onde se inclui os aspetos interativos de uma página.

O que é UX?

User experience (UX) é o conjunto de elementos relativos à interação do utilizador com um produto.
